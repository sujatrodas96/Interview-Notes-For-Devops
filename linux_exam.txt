1. Find Large Files Efficiently
You have a server running out of disk space.
Write a single command to find all files larger than 1GB in /var and
sort them by size (descending).

Ans: find /var -type f -size +1G -exec du -h {} + | sort -rh
         Find /var = searching in /var folder
        -type f = file type
        -size +1G = Greater than 1 GB
        -exec du -h = execute disk usage human readable format.
        sort -rh = sorting reverse human readable

4. Extract Specific Logs Efficiently
From a large log file (app.log), extract:
Lines
  containing "ERROR" 
But exclude
     lines containing "timeout" 
Show
     only last 100 matching entries

	Ans: grep "ERROR" app.log | grep -v "timeout" | tail -n 100

	grep -v = will not search  the pattern 
	grep = will search ERROR from app.log
	tail -n100 will search last 100 lines
	| = will pass after succeeding one command then pass the result to another command.

Disk Usage per User
8. How would you find disk usage per user in /home, sorted by usage size?


Ans: sudo du -sh /home/* | sort -rh

du -sh = disk utilities summarize and human readable format
/home/* = Target the all sub dir of  home 
sort -rh = sorting reverse human readable
