ImagePull Error vs CrashLoopBackOff
---------------------------------------

ImagePull Error: 
-----------------

Kubernetes cannot download the container image from registry.
---------------------------------------------------------------

Pod does not start at all

Common Reasons
--------------------
Wrong image name
Wrong tag
Private registry authentication issue
Network issue
Docker Hub rate limit

CrashLoopBackOff:
------------------------

Container starts successfully but crashes repeatedly.
--------------------------------------------------------

Pod goes into restart loop

Common Reasons
-------------------
Application error
Wrong environment variables
Port conflict
DB connection failure
Memory/CPU issue
